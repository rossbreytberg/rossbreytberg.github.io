Here are some of the things I've built.

Type to Voice Chat

Roommate asleep when you want to voice chat? No problem! This app turns what you type into audio that can be send to any output device you choose. Combine with a virtual microphone such as VB-Cable and you'll be able to talk to your friends on Skype, Discord, and more simply by typing. View on the Windows Store.

Polls

Like the name implies, Polls allows people to ask each other questions and both vote and comment on them. The question asker can set a color for each option in the poll and the comments are then color-coded based on the commenter's vote. It is also possible to filter out comments by poll option.

Arc3d.js

Show a 3d globe using webGL and map graph nodes and edges onto it with the simple api of Arc3d.js.

Board

Drag an image onto Board and it will appear not just on your screen but for everyone else who has it open. Click on images to enlarge them.

Pipe

Pipe is a file-sharing web application that streams data between users without saving it on the server. This one is due for a peer-to-peer remake now that WebRTC is out :)

Shot

Shot is a shooting game. Locate and shoot the red square as fast as you can.
